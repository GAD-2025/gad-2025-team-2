# Cursor에서 배포 서버에 바로 반영하기

## 🎯 목표

Cursor에서 코드를 수정하면 → 배포 서버에 바로 반영

---

## 🔧 해결 방법

### 방법 1: 배포 스크립트 만들기 (가장 쉬움!)

Cursor 터미널에서 한 번에 실행하는 스크립트:

```powershell
# 배포 스크립트 (deploy.ps1)
# 1. Git에 변경사항 커밋 및 푸시
# 2. 배포 서버에 SSH 접속
# 3. Git pull 및 서버 재시작
```

---

### 방법 2: Git Push 후 자동 배포 (권장!)

배포 서버에 자동 배포가 설정되어 있다면:
- Cursor에서 코드 수정
- Git commit & push
- 배포 서버가 자동으로 pull & 재시작

---

### 방법 3: Cursor 터미널에서 SSH 접속 문제 해결

Cursor 터미널이 입력을 받지 못하는 문제를 해결해야 합니다.
