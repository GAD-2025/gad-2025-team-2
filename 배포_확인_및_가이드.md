## 🔍 문제 해결

### 온보딩 정보가 마이페이지에 안 보이는 경우

**원인:**
1. 배포 서버에 코드가 반영되지 않음
2. 데이터베이스에 기본 매장이 생성되지 않음
3. API 엔드포인트가 작동하지 않음

**해결 방법:**

1. **배포 서버 확인:**
   ```powershell
   curl https://route.nois.club:3002/docs
   curl https://route.nois.club:3002/employer/stores/employer-test-001
   ```

2. **데이터베이스 확인:**
   - MySQL Workbench에서 `team2_db` 데이터베이스 확인
   - `stores` 테이블에 `is_main=True`인 레코드가 있는지 확인

3. **새로 회원가입:**
   - 기존 계정이 온보딩 시 기본 매장이 생성되기 전에 만들어진 경우
   - 새로운 계정으로 회원가입하여 테스트
