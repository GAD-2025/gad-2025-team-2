# 터미널 입력이 안 될 때 해결 방법

## 🔍 원인 확인

터미널에 입력이 안 되는 경우:

1. **다른 프로세스가 실행 중** (예: Vite 서버)
2. **SSH 접속 대기 중** (비밀번호 입력 대기)
3. **터미널이 포커스를 잃음**
4. **터미널이 멈춤**

---

## ✅ 해결 방법

### 방법 1: 새 터미널 열기 (가장 쉬움!)

**Cursor에서:**
1. 하단 **터미널 탭** 클릭
2. 터미널 탭 옆의 **"+" 버튼** 클릭
   - 또는 `Ctrl + Shift + `` (백틱 키)

**새 터미널이 열리면:**
- 프롬프트가 `PS C:\Users\...>` 형태로 보임
- 여기서 입력 가능!

---

### 방법 2: 현재 터미널 재시작

**현재 터미널이 멈춰있다면:**

1. **터미널 탭에서 휴지통 아이콘** 클릭 (터미널 종료)
2. **"+" 버튼** 클릭 (새 터미널 열기)

---

### 방법 3: Vite 서버가 실행 중인 경우

**Vite 서버가 실행 중이면:**
- 그 터미널은 서버가 사용 중
- **새 터미널을 열어서** SSH 접속하세요

---

### 방법 4: SSH 접속 중인 경우

**비밀번호 입력 대기 중이면:**
- 비밀번호를 입력하세요 (화면에 표시 안 됨)
- 입력 후 Enter

**접속이 안 되면:**
- `Ctrl + C` 눌러서 취소
- 다시 시도

---

## 🚀 올바른 순서

1. **새 터미널 열기** (`+` 버튼)
2. **SSH 접속 명령어 입력:**
   ```powershell
   ssh root@route.nois.club
   ```
3. **Enter 누르기**
4. **`yes` 입력** (호스트 키 확인)
5. **비밀번호 입력** (`team2pass`)
6. **Enter 누르기**

---

## ⚠️ 주의사항

- **Vite 서버가 실행 중인 터미널**은 그대로 두세요
- **SSH 접속은 새 터미널**에서 하세요
- 터미널이 멈추면 `Ctrl + C`로 취소 후 다시 시도
