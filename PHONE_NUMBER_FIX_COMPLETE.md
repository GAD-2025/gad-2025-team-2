## 🧪 테스트 방법

### 1. 고용주 온보딩 테스트
1. 고용주 회원가입 진행
2. 회사 정보 입력 단계에서:
   - 회사 이름 입력
   - 주소 입력
   - **전화번호 입력** (새로 추가됨)
   - 업직종 선택
3. 전화번호가 자동으로 포맷팅되는지 확인 (010-1234-5678)
4. 전화번호 없이 진행 시도 → 에러 메시지 확인

### 2. 공고 관리 페이지 테스트
1. http://localhost:5173/employer/job-management 접속
2. 등록된 공고 카드 확인
3. 전화번호가 표시되는지 확인 (📞 010-1234-5678 형식)

### 3. 매장 추가 테스트
1. 마이페이지에서 매장 추가
2. 전화번호 입력
3. 공고 등록 시 해당 매장 선택
4. 공고 관리 페이지에서 전화번호가 표시되는지 확인

## 📝 참고사항

- 전화번호는 하이픈 포함 형식으로 입력받지만, 저장 시 하이픈이 제거됩니다.
- 공고 관리 페이지에서는 `job.shop_phone`이 있으면 표시하고, 없으면 표시하지 않습니다.
- 고용주 온보딩에서 입력한 전화번호는 기본 매장(Store)의 전화번호로 저장됩니다.
